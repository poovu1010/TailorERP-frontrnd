# StitchFlow — Tailoring Order Management System

## The Story Behind It

My dad is a tailor. Every day he manages customer measurements,
order deadlines, and delivery tracking using pen and paper.
I watched him lose track of orders, forget measurements, and
manually follow up with every customer.

That's why I built StitchFlow — a simple system that does all
of that digitally. It's not just a project. It's something
I actually wanted to exist.

## What It Does
- Manage customers and their body measurements
- Track orders from intake to delivery
- Shop owner and customer role separation
- Full order lifecycle — create, update, complete, delete

## Tech Stack
**Backend:** Node.js, Express.js, MongoDB, Mongoose
**Frontend:** React.js, Context API, React Router, Tailwind CSS
